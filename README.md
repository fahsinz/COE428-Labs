This GitHub repository contains a collection of C programming projects completed for the COE 428: Engineering Algorithms and Data Structures course. 
Focuses on fundamental data structures, algorithmic analysis, and modular software development.

Lab 1: C Review & Test-Driven DevelopmentAn introduction to incremental development and command-line argument processing. Includes a modular sorting framework where a custom 
mySort() implementation is tested against hardcoded data and user-provided inputs via argc and argv.

Lab 2: Recursion & The Towers of HanoiAn exploration of recursive algorithms. Features a solver for the Towers of Hanoi that traces recursion depth and moves using stdout for results and stderr for debugging, demonstrating advanced stream redirection techniques.

Lab 3: Sorting Metrics & Performance AnalysisImplementations of Insertion Sort and Merge Sort using a metrics-driven framework. The project emphasizes algorithmic efficiency by tracking comparisons, swaps, and moves to verify theoretical O(n^2) and O(nlog n) complexities.

Lab 4: Finite State MachinesA command-driven state machine simulator (simState). It supports dynamic reconfiguration, reachability analysis to identify "garbage" states, and state deletion, utilizing a custom-generated state table.

Lab 5: XML-based Heaps & StacksA multi-part project involving structure validation and data organization. Includes an XML "well-formedness" checker using a Stack and a Max-Heap implementation that outputs its internal tree structure as XML and performs a Heap Sort.Key Concepts CoveredData Structures: Arrays, Stacks, Heaps, and State Tables.Algorithms: Recursion, Merge Sort, Insertion Sort, and Reachability.Systems Programming: Memory management, Makefile automation, and standard I/O redirection.
